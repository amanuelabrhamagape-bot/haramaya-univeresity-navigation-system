# 🚀 HOW TO RUN - Campus Navigator (PHP + MySQL)

## Step 1 — Start XAMPP
1. Open **XAMPP Control Panel**
2. Click **Start** next to **Apache**
3. Click **Start** next to **MySQL**
4. Both should show green "Running" status

## Step 2 — Install the Database
Open your browser and go to:
```
http://localhost/cns/install.php
```
You will see a green ✅ success screen.
This creates the database and fills it with all data automatically.

## Step 3 — Open the Website
```
http://localhost/cns/index.php
```
